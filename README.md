# RollerCoaster Tycoon

A browser park-management game inspired by RollerCoaster Tycoon, built with React, TypeScript, and HTML5 Canvas. The focus is grid-based building, guest simulation, and lightweight management systems.

## Highlights

- Build paths, rides, stalls, and scenery on a grid map
- Guest needs, park economy, breakdowns, and staff systems
- Save and load flow with a canvas-rendered world

## Stack

- React
- TypeScript
- Vite
- HTML5 Canvas

## Quick start

```bash
npm install
npm run dev
```

Build and preview:

```bash
npm run build
npm run preview
```
